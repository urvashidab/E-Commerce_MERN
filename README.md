A modern E-Commerce web application built with React.
This project demonstrates a clean component-based architecture, product browsing, and shopping cart functionality.

The goal of this project is to practice React fundamentals, state management, and reusable UI components while building a real-world application.

## **Tech Stack**

**Frontend**

- **React**
- **JavaScript**
- **HTML**
- **Tailwind**
- **React Router**
